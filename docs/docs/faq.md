Q: Can I use this with GitHub Pages?

A: Yes. For a project site set `baseurl: '/<repo-name>'` in `_config.yml`. For a user/organization site set `baseurl: ''`.

Q: How do I add search?

A: Small sites can use a client-side search (lunr.js) or a hosted provider (Algolia). That would be an easy next step.
